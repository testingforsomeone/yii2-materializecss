Class macgyer\yii2materializecss\widgets\media\Carousel

All Classes | Properties | Methods
Inheritancemacgyer\yii2materializecss\widgets\media\Carousel » macgyer\yii2materializecss\lib\BaseWidget » yii\base\Widget
Uses Traitsmacgyer\yii2materializecss\lib\MaterializeWidgetTrait

Carousel is a robust and versatile component that can be an image slider or an item carousel with arbitrary HTML content.

Simply provide the $items as an array of items. For each item you can define the tag and content key for general configuration. To make the item more accessible, specify options which will be rendered as HTML attributes for the item element.

'itemOptions' => [
     'class' => 'amber white-text' // this class will be used for all carousel elements
],
'items' => [
     [
         'content' => Html::img('http://lorempixel.com/800/800/sports/2'),
     ],
     [
         'content' => '<h2>Carousel item heading</h2><p>Arbitrary content</p>'
         'options' => ['class' => 'carusel-item-override'] // overrides $itemOptions
     ]
],
'fixedItemOptions' => [
     'tag' => 'p',
     'content' => 'Some content',
],

See also http://materializecss.com/carousel.html.

$fixedItemOptions public property

The configuration for the fixed item.

The following special options are recognized:

  • tag: the fixed item's HTML tag name
  • content: the content of the fixed item. Please note: this can be HTML and will not be encoded.

If you do not want the fixed item to be rendered, set this option to false.

See also http://materializecss.com/carousel.html#special.
